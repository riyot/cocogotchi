▸ pwnagotchi-esp32s3 · web flasher

flashes the latest firmware directly to your esp32-s3 via webusb. chrome / edge on desktop only.

before clicking install:

  1. unplug the esp32-s3
  2. hold the BOOT button
  3. plug usb-c while still holding BOOT
  4. release after 2 seconds — chip is now in download mode

your browser does not support webusb. use desktop chrome or edge. this page must be served over https. open the github pages url, not a local file.

builds are produced by github actions on every push to firmware/. the manifest.json served alongside this page always points to the latest .bin.

need to roll back ? grab a previous build from the workflow runs.
